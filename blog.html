Question-1: What are the different ways to select an element in the DOM?

Most common methods are: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Common difference are:
innerHTML: Gets or sets HTML content, including tags.
innerText: Gets or sets the visible text content, considering styles and visibility.
textContent: Gets or sets the raw text content, without HTML tags, including hidden content.

Question-3: What is event delegation in the DOM?

Event delegation is an efficient way to handle events on dynamically added elements or many elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event, triggered on a child element, propagates up to its ancestors in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating Elements: document.createElement()
Adding Elements: appendChild()
Removing Elements: remove()
